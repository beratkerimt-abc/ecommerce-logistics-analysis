# Brazilian E-Commerce Logistics Analysis

This project analyzes the logistics performance of a Brazilian e-commerce platform 
using real order data from 2016 to 2018.

## Dataset
- Source: [Kaggle - Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
- 100,000+ orders, 9 tables
- Tables used: orders, customers, order items, reviews

## Project Structure
- Data Loading & Merging
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Interactive Dashboard (Plotly)

## Key Findings
- Average delivery time is 12 days
- Most orders arrive earlier than estimated
- Early delivery strongly correlates with higher review scores
- Remote states (RR, AM, PA) have significantly longer delivery times
- Order volume grew consistently from 2016 to 2018

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Plotly
